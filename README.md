# finpilot
Backend system for personal finance management 

# STACK:

- *Framework*: FastAPI
- *ORM*: SQLAlchemy
- *Migrations*: Alembic
- *Database*: PostgreSQL
- *Containerization*: Docker & Docker Compose
- *API Docs*: Swagger UI (auto-generated by FastAPI)
